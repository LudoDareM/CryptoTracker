# CryptoTracker Pro

CryptoTracker Pro è una web app che permette di visualizzare in tempo reale i dati delle principali criptovalute, con grafici dinamici, ricerca istantanea e interfaccia moderna.

## 🚀 Funzionalità

- 🔍 **Ricerca criptovalute** con aggiornamento dinamico dei risultati  
- 📊 **Grafico storico** del prezzo (ultime 24 ore)  
- 💹 **Prezzo, variazione percentuale, market cap** e altri dati in tempo reale  
- ⚡ **Interfaccia veloce e reattiva**  
- 🎨 **UI moderna**, con gradienti, blur e animazioni  
- 🌐 **Deploy online** tramite GitHub Pages

## 🛠️ Tecnologie utilizzate

- **HTML5**
- **CSS3**
- **JavaScript**
- **Chart.js** per i grafici
- **CoinGecko API** per i dati crypto
- **GitHub Pages** per la pubblicazione

## 📡 API

L’app utilizza l’API pubblica di CoinGecko per ottenere:

- prezzo attuale  
- variazione 24h  
- market cap  
- grafico storico
